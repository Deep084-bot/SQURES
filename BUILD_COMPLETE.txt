CYCLOMATIC COMPLEXITY (CC):
   Measures code path complexity
   
   ✅ ≤ 6:     Low Risk - Simple, maintainable code
   ⚠️  7-10:    Medium Risk - Some complexity concerns
   ❌ > 10:     High Risk - Needs refactoring

MAINTAINABILITY INDEX (MI):
   Composite quality metric (0-100 scale)
   
   ✅ 85-100:   Excellent - Highly maintainable
   ✅ 70-84:    Good - Well-written code
   ⚠️  50-69:    Fair - Some maintenance concerns
   ❌ < 50:     Poor - Difficult to maintain

RISK LEVELS:
   🟢 Low:     Well-written, maintainable
   🟠 Medium:  Some quality concerns
   🔴 High:    Significant issues
   ⚪ Unknown:  Analysis error
